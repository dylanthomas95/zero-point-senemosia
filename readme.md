# Zero Point Senemosì̀a Framework

Framework principale per l'estrazione di energia latente nei sistemi organizzativi complessi.

## 🎯 Overview

Il framework Zero Point Senemosì̀a (ZPS) applica i formalismi della meccanica quantistica, dei difetti topologici e della termodinamica cognitiva allo studio delle organizzazioni complesse.

### Caratteristiche Principali

- **Spazio di Hilbert Organizzativo**: Modellazione degli stati organizzativi come qubit
- **Regolarizzazione di Ramanujan**: Stabilizzazione delle equazioni divergenti
- **Sezione Aurea (ϕ)**: Ottimizzazione geometrica della convergenza
- **Principio dell'Energia Libera Variazionale (FEP)**: Minimizzazione dinamica
- **Curva Frattale di Gosper**: Compressione informativa geometrica

## 📦 Installazione

```bash
pip install zps-framework
```

## 🚀 Quick Start

```python
from zps import Framework, OrganizationalSystem

# Inizializza il framework
zps = Framework()

# Crea un sistema organizzativo
org = OrganizationalSystem(n_agents=50)

# Estrai energia di punto zero
energy = zps.extract_zero_point_energy(org)
print(f"ZPE Potential Index: {energy.zpe_index}")
```
